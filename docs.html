Placeholder Guide ♡

Use placeholders in greetings, embeds, autoresponders, welcome messages, and custom server text.

USER PLACEHOLDERS

{user} {username} {displayname} {mention} {userid} {avatar}

SERVER PLACEHOLDERS

{server} {servername} {serverid} {membercount} {owner}

ROLE & DATE PLACEHOLDERS

{toprole} {rolecount} {date} {time} {created} {joined}

CHANNEL PLACEHOLDERS

{rules} {roles} {introductions} {chat} {help}

♡ Welcome {mention} to {server}!

Please read {rules}, grab roles in {roles}, and say hi in {chat}.
