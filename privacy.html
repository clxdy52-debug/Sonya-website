Privacy Policy

Sonya respects your privacy and only stores data required for bot functionality.

Data Collected

Sonya may store Discord user IDs, server IDs, role settings, profiles, economy data, levels, tickets, and configuration settings.

How Data Is Used

Stored data is used solely to provide bot features such as profiles, moderation, tickets, role panels, economy systems, and leveling.

Data Sharing

Sonya does not sell, rent, or share user data with third parties.

Data Security

Reasonable measures are taken to protect stored data from unauthorized access.

Data Removal

Server owners may request removal of stored server configuration data by contacting support.

Policy Updates

This Privacy Policy may be updated at any time without prior notice.
